# Kwick Player — Roku

Sideload Kwick Player on your Roku and see what's new in every version.

## Download

Grab the latest `KwickPlayer-Roku.zip` from the
**[Releases page](../../releases/latest)**.

## How to install (sideload)

Roku doesn't have this in the Channel Store, so you install it in
Developer Mode:

1. On your Roku remote press: **Home ×3, Up ×2, Right, Left, Right, Left, Right**
2. The Developer Settings screen opens — **enable Developer Mode**, accept
   the agreement, and note the **IP address** it shows (and set a password).
3. On a computer on the same Wi-Fi, open a browser to that **IP address**.
4. Log in (username `rokudev` + the password you set), choose
   `KwickPlayer-Roku.zip`, and click **Install**.
5. Kwick Player launches. Sign in with KwickTV, or add your own Xtream
   Codes / M3U provider.

## What it does

Live TV with a timeline guide, movies & series, favorites, and continue
watching — the Roku version of the Kwick Player app, matched to the
Android layout as closely as Roku allows.
